# Vin Kvalitet - Dataanalyse Projekt 
Dette projekt er en begyndervenlig dataanalyse af rød- og hvidvin. Formålet er at undersøge, hvilke kemiske egenskaber (som alkohol, sukker og syre), der giver den bedste vinkvalitet.

## Hvad gør denne Python?
* **Datarensning:** Fjerner tomme felter, dubletter og ekstreme tal (outliers).
* **Visualisering:** Laver flotte grafer, diagrammer og et farvekort (heatmap) for at finde sammenhænge i vinen.
* **Skalering:** Gør tallene nemmere at sammenligne for computeren.
* **Statistik & Test:** Undersøger vores gæt (hypoteser) – for eksempel om 14% alkohol giver den bedste vin, og om rødvin egentlig er stærkere end hvidvin.
